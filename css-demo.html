Lý thuyết CSS cơ bản

CSS dùng để mô tả cách phần tử HTML hiển thị. Cú pháp cốt lõi là selector { property: value; }.

1. Selector là gì

Selector chọn phần tử sẽ bị áp dụng style.

p { color: red; } .card { padding: 16px; } #main { width: 100%; }
  • Selector theo thẻ: p
  • Selector theo class: .card
  • Selector theo id: #main

2. Box model

Mỗi phần tử là một hộp gồm content, padding, border, margin.

width → kích thước content padding → khoảng đệm bên trong border → viền margin → khoảng cách bên ngoài

Demo bên phải đang tách rõ margin/padding để bạn quan sát trực quan.

3. Cascade và specificity

Khi nhiều rule cùng tác động, CSS sẽ ưu tiên theo độ cụ thể và thứ tự khai báo.

/* ưu tiên cao hơn */ #title { color: blue; } .title { color: green; } h1 { color: black; }

Thứ tự thường gặp: inline style > id > class > thẻ.

4. Display

Display quyết định phần tử chiếm chỗ và tham gia layout như thế nào.

  • block: chiếm cả dòng.
  • inline: nằm chung dòng.
  • inline-block: như inline nhưng có kích thước.
  • none: ẩn khỏi layout.
  • flex, grid: layout hiện đại cho cha-con.

5. Flex và Grid dùng khi nào

Flex hợp cho sắp hàng theo 1 chiều. Grid hợp cho bố cục 2 chiều.

.row { display: flex; flex-direction: row; } .grid { display: grid; grid-template-columns: repeat(3, 1fr); }

Trong playground, đổi display của vùng chứa để thấy khác biệt ngay.

6. Cách dùng CSS nhanh

  • Xác định phần tử cần chỉnh.
  • Chọn thuộc tính phù hợp.
  • Đặt giá trị rõ ràng, đo được.
  • Kiểm tra ảnh hưởng lên bố cục tổng thể.
.box { margin: 12px; padding: 16px; background: #ff5722; color: #fff; }
