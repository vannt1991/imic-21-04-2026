1. Mục tiêu

Nắm nền tảng JavaScript để xử lý dữ liệu đơn giản

Sau buổi học, học viên hiểu được cách khai báo biến, dùng điều kiện, lặp qua dữ liệu, viết hàm và thao tác với mảng, object trong các bài toán cơ bản.

Kiến thức

Nắm được biến, kiểu dữ liệu, toán tử và các cấu trúc điều khiển thường dùng.

Logic

Biết phân tích dữ liệu đầu vào, chọn điều kiện xử lý và tách bài toán thành hàm nhỏ.

Thực hành

Viết được các bài nhỏ như tính điểm trung bình, tìm phần tử lớn nhất và quản lý mảng sản phẩm.

2. Nội dung chính

Chia kiến thức theo từng khối dễ giảng và dễ nhớ

Phần này bám theo đúng mạch học của lớp: từ khai báo dữ liệu đến tư duy giải bài toán bằng JavaScript.

Biến, kiểu dữ liệu, toán tử

Giới thiệu let, const, số, chuỗi, boolean, so sánh và phép toán số học.

const name = "An";
let score = 8.5;
const isPassed = score >= 5;

if/else và switch case

Dùng điều kiện để rẽ nhánh theo đúng logic bài toán và chọn nhiều trường hợp với switch.

const day = 2;

if (day === 1) {
  console.log("Chủ nhật");
} else {
  console.log("Ngày thường");
}

Loop: for, forEach, for...in, while, do...while

Dùng vòng lặp để duyệt mảng, duyệt object và lặp theo điều kiện cho đến khi đạt yêu cầu.

for (let i = 0; i < 3; i++) {
  console.log(i);
}

Function

Tách logic thành hàm để tái sử dụng, kiểm thử và giải thích bài toán rõ hơn.

function greet(name) {
  return `Xin chào, ${name}`;
}

Array, Object

Làm việc với danh sách và dữ liệu có cấu trúc, đặc biệt là các bài quản lý sản phẩm và sinh viên.

const products = [
  { id: 1, name: "Áo thun", price: 120000 }
];

2.1. So sánh nhanh

for, forEach và while khác nhau thế nào?

Đây là phần chốt nhanh để học viên nhớ đúng công cụ cho từng tình huống. Có thể dùng như một slide chuyển ý trước khi sang thực hành.

for

Dùng khi cần lặp theo chỉ số, duyệt mảng có kiểm soát và có thể dừng giữa chừng.

  • Mạnh nhất khi cần index.
  • Dễ kết hợp break và continue.
  • Phù hợp cho học viên mới vì cú pháp rõ ràng.

forEach

Dùng khi chỉ cần duyệt từng phần tử của mảng và thực hiện hành động với phần tử đó.

  • Viết ngắn, dễ đọc.
  • Không dùng để thoát vòng lặp sớm.
  • Hợp với xử lý hiển thị, map dữ liệu, log danh sách.

while

Dùng khi chưa biết trước số lần lặp, chỉ biết điều kiện dừng.

  • Hợp với bài toán lặp theo trạng thái.
  • Cần cập nhật điều kiện cẩn thận để tránh lặp vô hạn.
  • Rất hữu ích khi làm bài tập logic cơ bản.
Gợi ý nhớ nhanh: cần index thì chọn for, chỉ đọc từng phần tử thì chọn forEach, chưa biết số lần lặp thì chọn while.

3. Thực hành trên lớp

Bốn bài nhỏ để học viên nhìn thấy logic chạy ngay

Bấm vào từng nút để xem kết quả minh họa. Nội dung tương ứng với các bài tập nền tảng trong buổi học.

Bài 1

Tính điểm trung bình

Từ một mảng điểm, cộng tổng rồi chia cho số lượng phần tử.

calculateAverage([8, 9, 7.5, 10]);

Bài 2

Kiểm tra chẵn/lẻ

Dùng toán tử chia lấy dư để xác định một số là chẵn hay lẻ.

isEven(17);

Bài 3

Tìm phần tử lớn nhất

Đi qua mảng số và tìm giá trị lớn nhất.

findMax([12, 7, 23, 9, 15]);

Bài 4

Thao tác mảng sản phẩm

Lọc, đổi trạng thái và định dạng danh sách sản phẩm để hiển thị ra màn hình.

formatProducts(products);

4. Bài tập về nhà

Luyện thêm để củng cố tư duy JavaScript cơ bản

Phần này có thể giao dưới dạng danh sách bài tập ngắn, học viên làm độc lập hoặc theo cặp.

10 bài luyện JavaScript cơ bản

  1. Khai báo biến lưu tên, tuổi và in ra câu giới thiệu.
  2. Tính tổng hai số và in kết quả.
  3. Kiểm tra một số là chẵn hay lẻ.
  4. Tìm số lớn nhất trong một mảng số.
  5. Tính tổng các phần tử trong mảng.
  6. Đếm số phần tử lớn hơn 10 trong mảng.
  7. Viết hàm đảo ngược một chuỗi.
  8. Lọc ra các số dương từ một mảng hỗn hợp.
  9. Đếm số lần xuất hiện của một ký tự trong chuỗi.
  10. Viết hàm quản lý danh sách sinh viên: thêm, sửa, xóa.

Hướng dẫn làm bài quản lý sinh viên

Bắt đầu bằng một mảng các object có id, name, score, sau đó tách ra ba hàm riêng cho thêm, sửa và xóa. Cách làm này giúp học viên luyện tư duy xử lý dữ liệu theo từng bước. 

const students = [
  { id: 1, name: "An", score: 8 },
  { id: 2, name: "Bình", score: 7.5 }
];
